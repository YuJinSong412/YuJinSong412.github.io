##27일차##

```
<회화>
지저분한 책상은 내 것이야.
-A desk which is messy is mine.
널찍한 책상은 네 것이야.
-A desk which is spacious is yours.
바닥에 있는 책상은 내 것이 아니야.
-A desk which is on the floor is not mine.
오래된 책상들은 네 것들이 아니야.
-Desks which are old aren't yours.
나는 오래되지 않은 책상을 가지고 있어.
-I have a desk which is not old.
내 방에 있는 책상들을 좋아하니?
-Do you like desks which are in my room?
비싼 차가 내 차 일수도 있어.
-A car which is expensive could be mine.
나는 비싼 차를 운전하곤 했었어.
-I used to drive the car which is expensive. 
서랍이 있는 책상이 더 좋다.
-A desk which has a drawer is better.
쉽게 망가지는 책상은 더 나빠.
-A desk which breaks easily is worse.
수업 늦게 시작하는게 더 좋지 않아.
-A class which starts late is not better.
일찍 시작하지 않는 수업은 더 나쁘지 않아.
-A class which doesn't start early is not worse.
문이 2개 있는 차들은 더 좋아.
-Cars which have 2 doors are better.
늦게 열지 않는 가게들은 더 나쁘다.
-Stores which don't open late are worse.
나는 늦게 여는 가게를 좋아해.
-I like a store which opens late.
요리 안하는 남자가 여기 있어.
-A man who doesn't cook is here.
나는 일찍 시작하지 않는 수업을 좋아해.
-I like a class which doesn't start early.
오래되어 보이지 않는 차들을 아니?
-Do you know cars which don't look old?

```
